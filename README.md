# To Do App

## About

This is an app simple for a to do task list.   
You can also use for list yours notes.   
Just write the note or the task and click the + button.   
Then will save with text and date.

![](.gitbook/assets/screenshot_1.jpg)

## Build With

* React Native
* Expo
* Nodejs v10
* Npm v6
